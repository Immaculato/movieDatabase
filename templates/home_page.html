About

This website implements a django powered movie database. Registering for this website allows you to:

  • Search for movies based on genre, title, movie crew
  • Add tags to movies
  • Add reviews to movies
  • Rate movies. Let us know what movies are your favorite!
If you are lucky enough to be a manager, you will be able to add, delete, and modify the movies we have on record!
